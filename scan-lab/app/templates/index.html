3D Scan Viewer Demo

scan-lab is a modular Flask demo project for browser-based 3D scan visualization. It is intentionally structured so selected modules can later be transferred into larger projects.

  • Flask backend using the application factory pattern
  • Prepared service layer for business logic
  • Dedicated viewer page with three.js integration placeholder
  • Separate folders for templates, static assets, and data
